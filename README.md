# Modular Houses

Статический сайт для продажи модульных домов.

## Локальный просмотр

Откройте:

- `preview.command`

или поднимите любой статический сервер в корне проекта.

## Публикация через GitHub Pages

1. Создайте пустой репозиторий на GitHub.
2. Добавьте удаленный репозиторий:
   - `git remote add origin https://github.com/<username>/<repo>.git`
3. Отправьте проект:
   - `git push -u origin main`
4. В GitHub откройте:
   - `Settings -> Pages`
5. Выберите:
   - `Deploy from a branch`
   - branch `main`
   - folder `/(root)`

После публикации сайт будет доступен по адресу:

- `https://<username>.github.io/<repo>/`
